# Design System — Internal Tests

## Direction
Personality: Precision & High Density  
Foundation: Cool neutral  
Depth: Borders-only (quiet separators, minimal shadows)

## Tokens
### Spacing
Base: 4px  
Scale: 4, 6, 8, 12, 16, 20, 24

### Radius
Scale: 6px, 10px

### Surfaces
Canvas: #ffffff  
Surface: #ffffff  
Border: rgba(0, 0, 0, 0.08)

## Patterns
### Filter Field
- Container: border + subtle surface
- Icon + input inline
- Label: uppercase, small, letter-spaced

### Table
- Sticky header/cells with solid background to prevent overlap artifacts
- Hover rows, compact density
- Boolean columns centered with a single emphasized checkmark

### Cards
- Header: label + title + small status cue
- Body: compact spacing, clear sectioning
